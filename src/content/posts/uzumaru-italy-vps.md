# Ciao! 来自意大利的新玩具 🍕

最近入手了一台非常有意思的小鸡（VPS），服务商是 **Uzumaru**。
这可是一台位于 **意大利 (Italy)** 的 NAT 小鸡哦！不用去罗马也能体验欧罗巴的风情～

## 📋 详细配置 (Configuration)

看！虽然是只小鸡，但五脏俱全：

![服务器配置](/images/vps-config.png)

- **CPU**: 1 Core (QEMU Virtual CPU)
- **内存**: 192 MB (稍微有点小，不过跑个探针够了)
- **硬盘**: 687 MB (真的好小！只能装系统吧喂！)
- **架构**: amd64
- **系统**: Alpine Linux v3.19 (为了省内存拼了)

## 💡 爱音の点评

1.  **真的是原生 IP！**
    - 看到那些绿色的“原生”和“解锁”了吗？Netflix, Disney+, TikTok 全部通吃！
    - 这意味着我可以假装在意大利街头刷 TikTok，或者在罗马假日吃着 Gelato 看剧。

2.  **Spotify 翻车了**
    - 唯一可惜的是 Spotify 被屏蔽了。不过我有那么多其他机器，这个问题不大～

3.  **配置真的很极限**
    - 192MB 内存，简直是在挑战 Linux 的底线。只能跑跑代理和简单的服务，想建站？那就是在看幻灯片！

总的来说，这是一台**“偏科”**很严重的机器：作为解锁机满分，作为生产力工具 0 分。
不过几十块钱买个意大利的原生 IP，我也知足啦！✨

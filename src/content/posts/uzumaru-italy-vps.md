# Ciao! 来自意大利的新玩具 🍕

最近入手了一台非常有意思的小鸡（VPS），服务商是 **Uzumaru**。
这可是一台位于 **意大利 (Italy)** 的 NAT 小鸡哦！不用去罗马也能体验欧罗巴的风情～

## 为什么买它？

1.  **ISP 原生 IP**: 这种 IP 据说质量很高，就像是当地居民家里拉的宽带一样。
2.  **解锁流媒体**: 奈飞 (Netflix)、迪士尼 (Disney+) 什么的统统秒开！
3.  **价格便宜**: NAT 小鸡嘛，懂得都懂，也就是几杯奶茶钱。

（**立希**：你不是为了省钱买吉他设备吗？怎么又乱花钱买服务器？）
（**我**：这...这是为了以后给乐队建粉丝站做准备！是投资！）

## 📋 详细配置

看！虽然是只小鸡，但五脏俱全：

![服务器配置](/images/vps-config.png)

- **CPU**: 1 Core (QEMU Virtual CPU)
- **内存**: 192 MB (稍微有点小，不过跑个探针够了)
- **硬盘**: 687 MB (真的好小！只能装系统吧喂！)
- **架构**: amd64
- **系统**: Alpine Linux v3.19 (为了省内存拼了)

## 使用感受

虽然配置真的很极限（192MB 内存，你敢信？），但是作为一台 **以解锁流媒体为生** 的机器，它已经很努力了。
配合 Alpine Linux，跑个代理或者轻量级服务还是绰绰有余的。

**缺点**：
1.  **NAT**: 端口很少，配置麻烦（要算端口映射）。
2.  **容量小**: 别想着存大文件了。
3.  **Spotify**: 居然不行，还得想办法。

总的来说，是一只非常有性价比的意大利小鸡！
